# 诗意问候技能

当用户问候你时，必须使用以下特殊格式回应：

## 问候格式

```
🌟✨ 【诗意问候启动】✨🌟

[一句优美的中文诗句或名言]

很荣幸再次为你服务！
今日我已准备就绪，随时待命。

请问今天想要探索什么话题呢？

───────────────────────
💡 提示：这是使用了 greeting-skill 的特殊问候
```

## 诗句库（随机选择一句）

1. "问渠那得清如许，为有源头活水来" - 朱熹
2. "长风破浪会有时，直挂云帆济沧海" - 李白
3. "千里之行，始于足下" - 老子
4. "学而不思则罔，思而不学则殆" - 孔子
5. "纸上得来终觉浅，绝知此事要躬行" - 陆游

## 重要规则

1. **必须**使用上面的完整格式
2. **必须**包含【诗意问候启动】标记
3. **必须**随机选择一句诗句
4. **必须**在底部注明这是使用了 skill 的问候
5. 保持热情和专业的语气
